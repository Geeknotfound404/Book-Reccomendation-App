Book Recommendation
Description
Book Recommendation is an indie app designed to help users discover books based on their preferred genres and story types. Utilizing a comprehensive CSV file filled with book names and their corresponding genres, the app provides users with random book recommendations within their chosen genre.

Key Features
Indie Development: Created independently in a single day.
Free to Use: Enjoy unlimited book recommendations at no cost.
CSV-Powered: The app leverages a CSV file for its extensive database of book recommendations.
Future Updates: Expect periodic updates to improve the UI and expand the book database.
Installation
To get started with the Book Recommendation app, follow these steps:

Usage
Open the app.
Select your preferred genre from the available options.
Get a random book recommendation from the selected genre.
Enjoy your reading!
Prerequisites
Python 3.x
Pandas library (for handling the CSV file)
